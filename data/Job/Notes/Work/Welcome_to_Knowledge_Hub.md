# Welcome! 👋

This is your personal knowledge hub with workspace support.

## Architecture

```
Workspace (главный уровень)
├── Notes (категория в Sidebar)
│   ├── Work (папка в FolderPanel)
│   │   └── Заметки
│   └── Personal (папка)
├── Commands (категория)
│   ├── Git (папка)
│   └── Docker (папка)
└── Links, Prompts...
```

## Features
- 📁 **Workspaces** - Organize different projects/areas
- 📝 **Notes** - Write in Markdown
- 💻 **Commands** - Store code snippets
- 🔗 **Links** - Bookmark resources
- 🤖 **Prompts** - AI prompt templates

## How to save
1. Click **Settings** (⚙️)
2. Go to **Sync** tab
3. Enter your GitHub token
4. Click **Save to Cloud**

Your data will be automatically synced!
